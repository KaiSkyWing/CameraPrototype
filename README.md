## 📝 プロジェクト概要

15名のチームで開発中のホラーアドベンチャーゲームの開発初期プロトタイプです。
開発プログラマーチームが2つに分かれてゲームの企画書をもとに制作し、ゲームのコアになるギミックやアイディアへの理解を深めるのが目的でした。
そのうちの1つのチームのリーダーとなり、アイディア出しをチーム4人ほどで、開発は私1人で行いました。


一週間で作ることを目的としていたため、大きなものは作れませんでしたが、メインとなる過去と現在に干渉するカメラというものを作り上げました。

動画リンク：https://youtu.be/udSWNP601yk

現在はUIを主に担当しています。

紹介リンク：https://dynamic-earl-6dc.notion.site/UI-2a33b0fa766d81ec9e3fc5e6b6742a58

## 🧩 実装した主な機能

* カメラへと切り替えると過去のものが見える。

→LayerをNew(現在), Old(過去), All(共通)と分け、現在と共通、過去と共通を写す2つのカメラを切り替えることで実装。
* ピントを橋に合わせ撮影することで過去の状態のものを復元できるように。また、再度撮影することで元通りにもなる。
* 撮影時フラッシュが出て変化が起きたことがわかりやすいように。
* PostProcessingを用いてカメラ液晶画面であったり色合いなどを変更しカメラを持っていることがわかりやすく工夫。

基本挙動は外から過去プロジェクト・外部などから引用
モデルはUnityAssetStoreから引用
